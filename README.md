# CS Projects

12 mini projects to build yourself, ordered by difficulty.
Each folder has:
- A `README.md` explaining exactly what to build
- A starter file with scaffolding and `TODO` markers
- A `tests.py` you can run to check your work

**How to use:**
1. Read the `README.md` in each folder
2. Fill in the TODOs in the starter file
3. Run `python3 tests.py` to see how many tests pass
4. Move on when all tests are green

No looking at solutions until you've given it a proper attempt.
